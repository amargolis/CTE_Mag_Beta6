A. Quick Setup - top

There is no setup. Just open the YouTubePlayer.xcodeproj and you will be on your way. 1 prerequisite is that you have XCode 4.4.1 installed one you mac.

B. Basic Configuration - top

The only thing you need to change the customize URL_DATA in YouTubeTableViewController.m file and customizing to YouTube.plist file (YouTubePlayer Supporting Files folder). Upload .plist file to your server.

C. Advanced Configuration - top

Adjustments
Add to your project
[1] Copy and paste the YouTubePlayer Supporting Files folder contents to your project, add the missing frameworks.
    Set -fno-objc-arc compiler flag to JSONKit.m file. JSONKit no ARC file.

[2] Copy and paste to your project YouTubeTableViewController .h+.m files and YouTubeCell .h+.m files.

[3] Copy and paste to your project storyboard files (Navigation Controller - YouTube Table View Controller).

[3] Customize URL_DATA in YouTubeTableViewController.m file.

[4] Customize YouTube.plis file, upload to your server.   

[5] Connect the Navigation Controller to your storyboard.
